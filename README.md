# Executive-sales-Dashboard
## Executive Sales Dashboard – Tableau

An interactive Executive Sales Dashboard built using Tableau to analyze and visualize sales performance across multiple years. This dashboard provides key insights into sales, profit, discounts, and quantity, helping business stakeholders monitor performance and identify trends.

## 📌 Project Overview

This project focuses on creating a business intelligence dashboard that summarizes key sales metrics and visualizes sales and profit trends from 2015 to 2018. The dashboard enables users to filter data dynamically and explore insights by product category, region, and customer segment.
The goal of the dashboard is to transform raw sales data into clear, actionable insights for executive decision-making.
